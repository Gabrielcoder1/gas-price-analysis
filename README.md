# gas-price-analysis
A data analysis project exploring gasoline price trends and their relationship to fuel spot prices from 2017 to 2026.
# Gas Price Analysis

## Project Overview

This project analyzes gas price data and related fuel market data to better understand price trends over time.

The project will start by cleaning and understanding one dataset at a time. After each dataset is cleaned and explored, useful datasets may be combined for deeper analysis.

## Current Datasets

- Gasoline retail prices
- Transportation fuel spot prices
- Weekly gasoline product supplied
- Transportation economic trends data

## Basic Plan

1. Set up the project structure
2. Add raw datasets
3. Clean and understand one dataset at a time
4. Explore patterns and trends
5. Combine datasets when useful
6. Add extra data only if needed
7. Create visuals and summarize findings
